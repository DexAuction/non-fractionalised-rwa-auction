# non-fractionalised-rwa-auction
Auction contracts for non-fractionalised RWA auctions

## Steps to compile and run
 
 ### to compile 
   ``` npx hardhat compile ```

### to run
   ``` npx hardhat run --network skale scripts/deploy.js ```

### to run test
  ```  npx hardhat test ```
